21 fractol project  
Main part:  
* Mandelbrot  
* Julia  
* Burning Ship  
Bonus part:  
* Koch-line  
* Koch's snowflake  
* Buddhabrot  
* Anti-buddhabrot  
Options:  
* Turn on|off Julia's motion  
* Zoom by mouse scrolling in certain screen's area  
* Increase|decrease fractal's power  
* Thread draw  
* Enable to change color in geometric fractals  
* Enable to move fractals with arrows
